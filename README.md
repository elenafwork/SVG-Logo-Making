  ## Description
  It generates the professional svg logo according to User's requirements in size 300x200 pixel image.
  ## Technology Used
  - JavaScript, 
  - NodeJS, 
  - Inquirer v8.2.4, 
  - SVG
  ## Installation
  Open terminal and type  
  ``` 
  node index.js
  ```
